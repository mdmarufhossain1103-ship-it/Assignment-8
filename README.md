# SunCart – Summer Essentials Store

# Description
A modern summer eCommerce platform where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more. Users can browse products, view details, and place orders after authentication.

## 🛠️ Technologies to Use

| Technology | Purpose |
| **Next.js** | Build the UI |
| **Tailwind CSS+Daisyui** | Styling and responsiveness |
| **BetterAuth** | Used for authentication |

# 3 key Features
☀️ 1. Seasonal Curated Collection
SunCart foucuses only on summer essentials, so customers don't waste time browsin irrelevant items. Everything-from sunglasses to beachwear-is handpicked for hot weather comfort and style.

⚡ 2. Fast & Easy Shopping Experience
A clean UI,Users can find what they need and complete view in just a click, even on mobile.

🎯 3. Personalized Recommendations
SunCart suggests products based on user preferences (like style, previous views, or trends), helping discover items that mathch their summer vibe.
